## What's new

Nordstar tracks per-package changelogs alongside its source. The release notes for the latest cut are on [GitHub Releases](https://github.com/NordcomInc/nordstar/releases) — every change shipped recently has been a pre-1.0 patch as we stabilize the API.

If you're upgrading and hit something unexpected, please [open an issue](https://github.com/NordcomInc/nordstar/issues).
