# Kbd

A small chip that represents a keyboard key or shortcut, rendered as a real `<kbd>` element.

## When to use

- To document a keyboard shortcut inline in prose or a tooltip.
- Inside menus, command palettes, and <C>Button</C> hints to show the key that triggers an action.

### When not to use

- For arbitrary inline code or tokens — use a `<code>` element instead.
- As an interactive control — `Kbd` is presentational; wire the actual shortcut separately.

## Accessibility

<Callout type="info" title="Use a real key">
    `Kbd` renders a native `<kbd>` element, which assistive tech announces as keyboard input. Keep the visible text equal to the actual key (e.g. `Esc`, `⌘`) so screen-reader users hear the same shortcut sighted users see.
</Callout>
