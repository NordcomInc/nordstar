Add `<Switch/>`, a boolean toggle built on `@radix-ui/react-switch` with `sm` and
`md` sizes and a `NordstarColor` checked state. It is keyboard operable,
form-associated via `name`/`value`, and supports both controlled and uncontrolled
use. Re-fitted to the Nordstar token contract.
