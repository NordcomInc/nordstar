Add `<Accordion/>`, a compound disclosure built on `@radix-ui/react-accordion`
(`Accordion.Item`, `Accordion.Trigger`, `Accordion.Content`). Supports `type="single"`
(with `collapsible`) and `type="multiple"`, with roving-focus keyboard navigation and a
chevron that rotates on open. Icon inlined (no icon dependency) and re-fitted to the
Nordstar token contract.
